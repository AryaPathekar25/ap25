# Social Media Investigator Tool

## 📌 Overview
This project automates social media data extraction using Selenium and OCR.

## 🚀 Features
- Automated scraping
- OCR-based text extraction
- GUI using Tkinter

## 🛠 Tech Stack
- Python
- Selenium
- OCR
- Tkinter

## 📊 Results
- Reduced manual effort by 60%
- Improved efficiency by 2x

## ▶️ How to Run
1. Install dependencies
2. Run main.py
